# Teftr

Personal finance tracker - Expense, income, and budget management web application.

## Overview

Teftr is a full-stack financial tracker built with Django and React, designed to help users manage their finances across multiple accounts and currencies.

## Technology Stack

- **Backend**: Django + Django REST Framework
- **Frontend**: React
- **Database**: PostgreSQL
